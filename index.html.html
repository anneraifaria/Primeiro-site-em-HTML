O que são Clients e Servers?

Clients: Dispositivos que consomem a informação da internet (computadores/notebooks/smartphones etc)"lado que consome".

Servers: Transferem informações para os clients.

Servidores são divididos em duas partes:

  • Softwares (parte lógica): SO, Monitoramento, Servidores Web
  • Hardware: Armazenamento, memória, processadores.

    • Tipos de Servidores:

  • Arquivos
  • Segurança (Firewall)
  • Streaming
  • E-mail
  • Web

Algumas definições:

  • Servidor Proxy: bloqueio de alguns sites, normalmente por redes empresariais. Pode ser servidor de validação. Problemas: proxy pode bloquear um retorno html por exemplo.
  • Firewall: Barreira de segurança para acessos externos, arquivos externos.

Considerações sobre Aplicações Web

Lembrar sempre que Aplicações Web NÃO são Bancos de dados!

Aplicações Web possuem a internet como meio de comunicação. Não é necessário a instalação no computador. Sem um app está instalado ele deixa de ser uma aplicação Web.

Facebook. É aplicação web no navegador, deixa de ser quando o app é instalado. A info é a mesma, mas a forma que foi programado é diferente. (Voltar)

Hospedagem de Sites

O que é uma hospedagem de sites?

Colocar os arquivos do site em um servidor.E como fazer isso?

  • Realizar o cadastro do domínio no registro.br (ou outro): Verificar se está disponível, lembrando que há um custo.
  • Se há uma ideia de marca, site, é recomendado registrar para garantir o domínio, sem o site pronto.
  • O DNS pode ser configurado depois.
  • Registro Br é exclusivo para sites.br.
  • Configura o acesso para o servidor DNS correspondente.
  • Deve-se observar se o servidor é multiplataforma, se hospeda com certificado SSL (conexão segura)

Muitas vezes o próprio servidor pode fornecer o serviço de emails.

Para enviar o conteúdo para o servidor, é utilizada a Conexão FTP (File Transfer Protocol)>, que é uma conexão direta com o servidor. Ex: Filezilla O provedor fornece usuário e senha.

Ferramentas e estruturas básicas do HTML

A estrutura primordial do HTML é composta por 3 partes:

  • Head: informações gerais sobre o documento, incluindo título.
  • Title: O título que fica na aba do navegador.
  • Body:Tudo que o usuário vai visualizar na página.

Polêmica: HTML NÃO É LINGUAGEM DE PROGRAMAÇÃO! É um arquivo de marcação!!! É um arquivo texto, delimitado, onde cada delimitação envia uma mensagem que o navegador consegue interpetar.

Listas Ordenadas e Não Ordenadas

O que são?

São elementos usados para identificar uma lista.

O elemento OL representa uma lista de itens ordenados. Geralmente os itens desse tipo de lista são mostrados com alguma contagem, que pode ser de qualquer tipo.

Já o elemento UL representa uma lista de itens sem ordem definida.
